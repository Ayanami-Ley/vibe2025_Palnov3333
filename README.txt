Инструкция по запуску:
1. Установите зависимости:
npm install
2. Создайте базу данных и таблицы (выполните db.sql в MySQL)
3. Запустите сервер:
node app.js
4. Откройте в браузере:
http://localhost:3000
Как получить Telegram Chat ID:
1. Создайте бота через @BotFather в Telegram
2. Начните диалог с ботом
3. Отправьте команду /start
4. Бот ответит с вашим Chat ID (или используйте бота @userinfobot)
